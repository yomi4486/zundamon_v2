【使用方法】
DockerとDocker-composeがインストールされた環境で「run.bat」を実行すればビルドから実行までやってくれます。
「.env」ファイルにご自身のBotのTokenやApplicationIDをご記入ください。
（隠しファイルになっている場合があり忘れることがありますが、書き忘れるととBotが起動しません。）

【ご利用にあたって】
Botを第三者に使用させる場合は、VOICEVOXの利用規約に則って利用するよう遵守させてください。
必ずBotの自己紹介、ドキュメントなどの目立つ場所に、「VOICEVOX：ずんだもん」と記述してください。

NVIDIAのGPUが使用可能な環境ならどなたでもご利用いただけます。
CUDAを使用可能にすると、より一層レスポンス速度を向上させることができます。
